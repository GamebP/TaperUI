# TaperUI - Multi-Game Hub

TaperUI is a multi-game utility script hub. It features a dark-mode layout, hotkey toggling, a dynamic games directory, and custom script integrations.

---

## Execution Guide

To run TaperUI, copy the script below, paste it into any Roblox executor, and click Execute.

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/GamebP/TaperUI/refs/heads/main/UI.lua"))()
```

### Step-by-Step Instructions:
1. Launch Roblox and join Ultimate Place Roulette or other supported games.
2. Open your executor.
3. Copy the loadstring above.
4. Paste it into your executor's text editor.
5. Click the Execute button.

---

## Key Features

* **Fast Loading:** Non-fullscreen opening transition.
* **Hotkey Toggle:** Press **`K`** to hide/show the interface (customizable in Settings).
* **Sleek Layout:** Snappy tab transitions without laggy delay.
* **Toast Notifications:** Smooth popup alerts at the bottom-right for system events.
* **Games Directory:** Browse supported places, search by name, and teleport on click.
